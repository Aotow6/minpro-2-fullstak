Contoh Properti CSS Display

Penjelasan

Halaman ini mendemonstrasikan bagaimana properti display dalam CSS mengubah cara elemen ditampilkan di halaman.

Display: Block

Elemen block selalu dimulai dari baris baru dan mengambil lebar penuh yang tersedia.

Elemen ini menggunakan display: block;
Elemen ini juga menggunakan display: block;

Display: Inline

Elemen inline tidak dimulai dari baris baru dan hanya mengambil lebar sebanyak yang diperlukan. Lebar dan tinggi tidak bisa diatur.

Elemen ini menggunakan display: inline;

Display: Inline-Block

Elemen inline-block mirip inline (tidak memulai baris baru), tetapi bisa diatur lebar dan tingginya seperti block.

Elemen ini
menggunakan
display: inline-block;

Display: None

Elemen dengan display: none; akan benar-benar dihilangkan dari tampilan dan tidak memakan ruang.
